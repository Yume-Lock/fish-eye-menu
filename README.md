# fisheye menu

interactive fisheye menu for nielsen's 10 usability heuristics

## tech stack

- react 19
- typescript
- vite
- framer-motion (animations)

## setup

```bash
npm install
```

## run

```bash
npm run dev
```

open http://localhost:5173

## build

```bash
npm run build
```

## features

- hover over menu items to see them enlarge (fisheye effect)
- tooltip displays the full heuristic name
- smooth animations and transitions
- responsive design
